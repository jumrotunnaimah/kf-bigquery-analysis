## Gambaran Umum Proyek
Proyek ini merupakan analisis data penjualan Kimia Farma menggunakan Google BigQuery sebagai bagian dari Final Task Rakamin Academy program Big Data Analytics. Analisis dilakukan dengan mengolah dan menggabungkan data transaksi, produk, serta kantor cabang untuk menghasilkan informasi seperti nett sales, persentase gross laba, dan nett profit yang dapat digunakan sebagai dasar evaluasi kinerja dan pengambilan keputusan bisnis.
