Kimia Farma BigQuery Analysis

## Deskripsi Proyek
Proyek ini merupakan bagian dari Final Task Rakamin Academy Data Science.
Analisis dilakukan menggunakan Google BigQuery untuk mengolah data transaksi
Kimia Farma dan menghasilkan tabel analisis yang siap digunakan untuk dashboard
dan pengambilan insight bisnis.

## Dataset
Dataset yang digunakan terdiri dari:
- kf_final_transaction
- kf_kantor_cabang
- kf_product
- kf_inventory

Seluruh dataset diolah dan digabungkan menggunakan SQL di BigQuery.

## Proses Analisis
Proses yang dilakukan dalam proyek ini meliputi:
- Querying dan penggabungan data transaksi, cabang, dan produk
- Perhitungan nett sales setelah diskon
- Penentuan persentase gross laba berdasarkan harga produk
- Perhitungan nett profit
- Penyusunan tabel analisis akhir

## Tools
- Google BigQuery
- SQL
- GitHub

## Output
Hasil akhir dari proyek ini adalah tabel analisis Kimia Farma
yang digunakan sebagai dasar pembuatan dashboard dan data storytelling.
